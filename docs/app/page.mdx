## Overview

Notiq is designed to be the go-to editor solution for Next.js developers who need a robust, feature-rich editing experience without the complexity of traditional WYSIWYG editors. Built on top of Lexical, Notiq offers a plugin-based architecture that makes it easy to extend and customize to your specific needs.

## Use Cases

Notiq is perfect for:

- **Content Management Systems** - Build rich content editors for blogs, documentation, and knowledge bases
- **Note-taking Applications** - Create powerful note-taking experiences with markdown and rich formatting
- **Collaborative Editing** - Implement real-time collaborative editing features
- **Documentation Sites** - Build interactive documentation with code examples and live previews
- **Admin Dashboards** - Add rich text editing capabilities to admin panels
